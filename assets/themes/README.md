# Темы Compact

Каждая тема хранится в отдельном JSON-файле. Встроенные темы находятся рядом с
этим файлом. Пользовательские темы можно положить в каталог `themes` внутри
папки настроек Compact; после следующего открытия настроек они появятся в списке.

Минимальный пример темы:

```json
{
  "id": "violet",
  "name": "Violet",
  "base": "dark",
  "colors": {
    "main": {
      "app_bg": "#210c31",
      "panel_bg": "#2d1740",
      "text_primary": "#f8efff"
    },
    "dialog": {
      "accent": "#a855f7"
    }
  }
}
```

`base` принимает значение `dark` или `light`. Неуказанные цвета наследуются от
соответствующей встроенной темы. Полный список доступных ключей можно посмотреть
в `dark.json` и `light.json`.
